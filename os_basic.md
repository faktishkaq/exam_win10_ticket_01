# Операционная система нужна для того что бы управлять компьютером, памятью и распределять приоритет для программ. 

- Для пользователя она показывает понятный интерфейс, защищает файлы, защищает его компьютер 
- Для программ - делит процессор, память, дает доступ к файлам

## GUI - графический интерфейс, удобен для повседневных задач и для того что бы не запоминать команды
## CLI - текстовые команды, удобен для выполнения одной команды и автоматизации 

# Примеры 
- GUI: Проводник, параметры
- CLI: CMD, PowerShell
